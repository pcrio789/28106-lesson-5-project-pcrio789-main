Briefly explain the difference between the generic font families.

CSS generic font families include serif, sans-serif, monospace, cursive, and fantasy. Serif fonts have designs at the ends of the letters, but sans-serif does not. Monospace fonts look very basic, same size etc. like computer code looks. Cursive font looks like hand writing and fantasy fonts are silly, decorative kind of fonts.

Briefly explain the different ways that you can indicate color on a web page or in text.

CSS defines colors on a web page using name colors like blue, red etc., RGB values, HSL Values and hexadecimal values. They offer different ways to design text, colored backgrounds and borders that make the pages look nice.
